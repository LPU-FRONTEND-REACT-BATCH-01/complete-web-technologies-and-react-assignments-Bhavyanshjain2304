JSX (JavaScript XML) — Quick Notes

• JSX = JavaScript XML
• A syntax extension for JavaScript
• Used mainly in React
• Lets you write HTML-like code inside JavaScript
• Makes UI code clean, readable, and easy to write


Example:
const element = <h1>Hello</h1>;


Converted into JavaScript:
React.createElement("h1", null, "Hello");


Key Rules:
• Must return a single parent element
• Tags must be properly closed
• Use {} for JavaScript expressions
• Use className instead of class
• Use camelCase for attributes
  Example: onClick, backgroundColor


Example with JavaScript:
const name = "Adi";
<h1>Hello {name}</h1>


Why JSX?
• Easier UI design
• Better readability
• Mixes UI and logic together
• Faster React development


One-line Exam Definition:
JSX is a syntax extension of JavaScript that allows writing HTML/XML-like elements inside JavaScript code.
